# **App Name**: Deen Journey

## Core Features:

- Landing Page: Display a welcoming message, emphasizing that 'Your Deen Matters' and offering entry points like 'Begin My Journey!' or 'Learn More'. Also emphasize it is free.
- Age and Language Selection: Request user's age; if under 10, prompt parental guidance. Also, ask for language selection, which configures the Deen Buddy.
- Difficulty Selection: Enable users to choose their level of depth into topics such as Salah, Surahs and overall understanding of Deen.
- Salah Tutorial: Provide a step-by-step tutorial on Salah (prayer), including visuals for each step and guidance on Wudu (ablution).
- Surah Access: Grant easy access to Surahs (chapters) from the Quran, categorized by difficulty level.
- Deen Improvement: Provide guidance on improving one's understanding and practice of Islam.
- Deen Buddy: Integrate an AI-powered assistant where users can ask questions about Islam, with the tool automatically providing relevant Hadiths, Surahs, and stories of the Prophets. Auto-add prayer times to home page based on location

## Style Guidelines:

- Primary color: Deep teal (#008080) to evoke serenity and spirituality.
- Background color: Very light teal (#E0F8F8), nearly white, for a calm, uncluttered feel.
- Accent color: Muted gold (#B8860B) for highlighting important elements and adding a touch of reverence.
- Body and headline font: 'Alegreya', a serif font, for a timeless, classic look. Note: currently only Google Fonts are supported.
- Employ clean, simple icons that are easy to understand, aiding navigation and representing different topics.
- Use a clean and intuitive layout with clear sections for each topic, ensuring ease of navigation and a user-friendly experience.
- Incorporate subtle animations for user feedback and to add a layer of interactivity without being distracting.
